# 基于Python的图书馆管理系统 python268

## 项目概述

本项目是一款基于Python语言开发的图书馆管理系统，系统采用MySQL数据库进行数据存储，并通过tkinter库构建图形用户界面（GUI），实现了图书馆的日常管理工作。

## 技术栈

- 后端：Python
- 数据库：MySQL
- 前端：tkinter

## 功能模块

### 用户认证功能

- **登录**：用户输入用户名和密码进行身份验证。
- **注册**：新用户进行用户名和密码的注册。

### 图书管理功能

- **添加图书**：管理员添加图书信息至数据库。
- **查询图书**：支持精确和模糊搜索。
- **更新图书信息**：修改图书信息并保存至数据库。
- **删除图书**：从数据库中删除指定图书记录。
- **导出图书数据**：将图书数据导出为CSV文件。

### 读者管理功能

- **添加读者**：添加新读者信息至数据库。
- **查询读者**：支持精确和模糊搜索。
- **更新读者信息**：修改并更新读者信息。
- **删除读者**：从数据库中删除指定读者记录。
- **导出读者数据**：将读者数据导出为CSV文件。

### 借阅与归还功能

- **借阅图书**：完成图书借阅流程。
- **归还图书**：处理图书归还操作，并记录逾期费用。
- **查询借阅记录**：查看借阅记录。
- **导出借阅记录数据**：将借阅记录导出为CSV文件。

### 逾期费用管理功能

- **查询逾期费用**：查看读者的逾期费用记录。
- **缴纳罚款**：更新罚款支付状态。
- **导出逾期费用数据**：将逾期费用数据导出为CSV文件。

### 其他功能

- **数据排序**：支持列表数据排序。
- **界面主题设置**：使用ttk.Style设置界面主题。
- **系统退出确认**：确保在退出前断开数据库连接。

## 系统角色

- 管理员：负责图书和读者信息的维护。
- 读者：进行图书借阅和归还。
